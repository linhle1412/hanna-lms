# CRM POS - Cosmetic Retail Prototype

## 🚀 Cách Sử Dụng

1. Mở file `index.html` bằng trình duyệt (Chrome/Edge)
2. Click vào sản phẩm để thêm vào giỏ hàng
3. Điều chỉnh số lượng bằng nút +/-
4. Click "THANH TOÁN" (hoặc F12) để thanh toán
5. Nhập số tiền và hoàn tất đơn hàng

## ⌨️ Phím Tắt

- `F1` - Đơn hàng mới
- `F2` - Tìm khách hàng
- `F3` - Focus ô tìm kiếm
- `F12` - Thanh toán
- `ESC` - Hủy/Đóng

## ✨ Tính Năng

- Tìm kiếm và lọc sản phẩm
- Giỏ hàng realtime
- Khuyến mãi tự động (giảm 10% khi ≥ 500k)
- Quà tặng tự động (mua 2 makeup tặng mascara)
- Tìm kiếm khách hàng
- Tích điểm loyalty
- Thanh toán đa phương thức
- Tính tiền thừa tự động
