# LMS Prototype - Styling Guide

## Color Palette

Based on your screenshots, here are the colors I've used in the prototype:

### Primary Colors
- **Orange Accent:** `#f17c23` (Primary buttons, active states, highlights)
  - Hover: `#d3681f`
  - Used for: Active nav items, primary buttons, links, section indicators

- **Dark Grey Sidebar:** `#2c3e50` (Main sidebar background)
  - Hover: `#34495e`
  - Used for: Sidebar navigation background

### Neutral Colors
- **White:** `#ffffff` (Main content, cards, modals)
- **Light Grey Background:** `#f8f9fa` (Page background)
- **Border Grey:** `#e0e0e0` (Borders, dividers)
- **Text Dark:** `#2c3e50` (Headings, primary text)
- **Text Medium:** `#555` (Secondary text, table headers)
- **Text Light:** `#666` (Tertiary text, breadcrumbs)

## Typography

### Font Family
- System fonts: `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif`

### Font Sizes
- **Page Title:** `28px` (Bold, weight 600)
- **Section Headers:** `20px` (Bold, weight 600)
- **Body Text:** `14px` (Regular, weight 400)
- **Small Text:** `12px-13px` (Status badges, labels)

### Font Weights
- **Bold:** `600` (Headings, important text)
- **Medium:** `500` (Labels, buttons)
- **Regular:** `400` (Body text)

## Component Styling

### Sidebar Navigation
- Background: `#2c3e50`
- Active item: Orange background (`#f17c23`)
- Logo: White background with black text
- Icon size: `18px`, spacing: `15px` margin-right

### Buttons
- **Primary:** Orange background (`#f17c23`), white text, `10px 20px` padding
- **Secondary:** White background, orange border and text, same padding
- Border radius: `4px`
- Hover effects: Darker shade transition

### Tables
- Header background: `#f8f9fa`
- Row hover: Light grey background
- Borders: `1px solid #e0e0e0`
- Cell padding: `15px`

### Modals
- Background: White with `8px` border radius
- Overlay: `rgba(0,0,0,0.5)` with blur effect
- Max width: `800px`
- Section indicators: Light orange background (`#fff5f2`)

### Input Fields
- Border: `1px solid #ddd`
- Padding: `10px 12px`
- Focus: Orange border (`#f17c23`)
- Border radius: `4px`

### Status Badges
- Rectangular boxes (not rounded pills)
- Padding: `4px 12px`
- Border radius: `3px` (minimal rounding)
- Background and text color vary by status

## Spacing

### Padding
- Content wrapper: `30px`
- Modal header/body/footer: `20px`
- Table cells: `15px`
- Sidebar nav items: `14px 20px`

### Margins
- Page title bottom: `20px`
- Section spacing: `20-25px`
- Form group spacing: `20px`

### Border Radius
- Cards/Containers: `4px`
- Modals: `8px`
- Buttons: `4px`
- Input fields: `4px`
