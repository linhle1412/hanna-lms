Configure date validation rules that will be applied during course creation and editing. These rules ensure:

  • Courses are scheduled with appropriate advance notice
  • MOF exam dates fall within acceptable timeframes
  • AOL exam periods are properly aligned with course dates

Note: Only one active configuration can exist at a time. The most recent active configuration will be used for new courses.

Business Rules Information

üìã How Rules Are Applied

  • Rule Versioning: Rules are applied at course creation time. Future changes won't affect existing courses.
  • Rule Priority: If multiple active configurations exist, the most recent one is used.
  • Default Values: If no active configuration exists, system uses defaults (2 days, 30 days, 3 days).
  • History Tracking: All changes are logged with user details and timestamps.
